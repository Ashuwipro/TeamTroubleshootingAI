# WebSeries Wire DOM XML Frontend Module

This folder contains frontend module files for the `WebSeries Wire DOM XML` payment screen.

- `ui.js`: Form UI rendering for the DOM payment screen.
- `actions.js`: Preview/generate request handling for the DOM payment screen.

These files are served by backend route `/payment-screens/webseries/wire-dom-xml/<filename>` and loaded by `backend/static/index.html`.

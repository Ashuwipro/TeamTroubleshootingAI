# WebSeries Wire INTL XML Frontend Module

This folder contains frontend module files for the `WebSeries Wire INTL XML` payment screen.

- `ui.js`: Form UI rendering for the INTL payment screen.
- `actions.js`: Preview/generate request handling for the INTL payment screen.

These files are served by backend route `/payment-screens/webseries/wire-intl-xml/<filename>` and loaded by `backend/static/index.html`.

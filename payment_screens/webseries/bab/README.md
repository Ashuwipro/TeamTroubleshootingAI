# WebSeries BAB Frontend Module

Project-level module folder for WebSeries BAB frontend code under `payment_screens/webseries/bab`.

- `ui.js`: Builds and wires the WebSeries BAB form UI.
- `actions.js`: Handles WebSeries BAB preview/generate API requests.

These files are served by backend route `/payment-screens/webseries/bab/<filename>` and loaded by `backend/static/index.html`.

# CHECKS XML Frontend Module

PCM-specific frontend module for `CHECKS XML` under `payment_screens/pcm/checks_xml`.

- `ui.js`: Renders the CHECKS XML form UI.
- `actions.js`: Handles preview and generate requests for CHECKS XML.

Assets are served via `/payment-screens/pcm/checks_xml/<filename>` and loaded by `backend/static/index.html`.

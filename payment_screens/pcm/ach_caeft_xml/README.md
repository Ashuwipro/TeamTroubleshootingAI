# ACH CAEFT XML Frontend Module

PCM-specific frontend module for the `ACH CAEFT XML` payment screen under `payment_screens/pcm/ach_caeft_xml`.

- `ui.js`: Renders the ACH CAEFT XML form fields.
- `actions.js`: Handles preview and generate requests for ACH CAEFT XML.

These assets are served by `/payment-screens/pcm/ach_caeft_xml/<filename>` and loaded by `backend/static/index.html`.
